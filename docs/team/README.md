# Documentación del equipo — Gestioname v2

Bienvenida al equipo de Datarecover. Esta carpeta contiene todo lo necesario para
empezar a probar y usar Gestioname durante la beta interna.

## Por dónde empezar

1. **[acceso.md](acceso.md)** — URLs, credenciales por rol y cómo arrancar en local.
2. **[modulos.md](modulos.md)** — Qué hace cada módulo y quién lo usa.
3. **[casos-de-prueba.md](casos-de-prueba.md)** — Checklist de pruebas por rol (marca lo que verifiques).
4. **[bugs.md](bugs.md)** — Plantilla para reportar bugs que encuentres.
5. **[arquitectura-resumen.md](arquitectura-resumen.md)** — Stack y comandos útiles (para perfiles técnicos).
6. **secretos.md** — Credenciales de producción (NO está en el repositorio; pídeselo a Julio).

## Cómo colaborar en la beta

- Usa la app con normalidad como si fuera tu día a día.
- Cuando algo falle o te chirríe, **anótalo** siguiendo la plantilla de [bugs.md](bugs.md).
- Prioriza los flujos de tu rol (ver [casos-de-prueba.md](casos-de-prueba.md)).
- Dudas → canal interno del equipo o info@datarecover.es.
